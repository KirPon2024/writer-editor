## SCOPE
Этот документ фиксирует завершение этапа A3 (SPEC HARDENING) и замораживает базовые OPS-инварианты CONTOUR-A.

A3 считается завершённым только при наличии отдельного OPS-документа DONE (этот файл).

## FROZEN_INVARIANTS (A1–A3)
- Фазы CHECK для write-задач: PRE_* до write-действий; POST_* после.
- DENYLIST обязателен и имеет приоритет над ALLOWLIST (default-deny).
- TYPE обязателен в каждом HARD‑ТЗ; допустимые значения фиксированы.
- Версионирование OPS-канона: CANON_VERSION и CHECKS_BASELINE_VERSION обязательны; новые инварианты не применяются ретроактивно.
- OPS-GATE (E0) обязателен для новых HARD‑ТЗ MODE A до исполнения.

## READ_ONLY
- Документы и правила A1–A3 после завершения считаются read-only.
- Любые новые OPS-инварианты допускаются только через новый этап (A4+).
- Запрещено требовать ретроактивного применения новых CHECK/инвариантов к существующим задачам.

## Summary (1–3 bullets)
- Репозиторий словарей для spellcheck (формат Hunspell: `.aff`/`.dic`).
- Полезно для будущего оффлайн spellcheck, но **у каждого словаря свой license**.
- Можно брать как источник словарей/языков, если соблюдём лицензии и объём.

## Relevant parts (where to look)
- Каждый словарь лежит в своей папке и содержит `LICENSE`.
- Искать нужные языки по коду (например `en_US`, `ru_RU` и т.п.).

## MVP Now (что можно взять прямо сейчас)
- Ничего обязательного (если spellcheck не в MVP) — только зафиксировать источник.

## Later (идеи на будущее, без внедрения сейчас)
- Встроенный оффлайн spellcheck: выбор языков + поставка словарей (или скачивание пользователем).

## License / usage notes
- Можно ли копировать код? Да (MIT для репо), но **файлы словарей** могут быть под другими лицензиями.
- Нужна отдельная проверка лицензий выбранных словарей + способ отображения атрибуции.

## Risks / pitfalls
- Размер: словари могут сильно увеличить приложение.
- Лицензии: у некоторых словарей ограничения на распространение.

## How to adapt in Craftsman (конкретно)
- Если добавим spellcheck: хранить словари локально, без сети; дать пользователю выбор языка.
- Указать лицензию/атрибуцию в `About`/`Licenses`.

## Summary (1–3 bullets)
- Классический “структурный” редактор документов (не plain text), сильный по модели документа и экспорту.
- Полезен только как продуктовый/UX‑ориентир (структура, экспорт, навигация).
- GPL → код не копируем.

## Relevant parts (where to look)
- Смысл больше в UX/концепции, чем в переносе кода.

## MVP Now (что можно взять прямо сейчас)
- Практически ничего (другой класс приложения), максимум — идеи навигации/структуры.

## Later (идеи на будущее, без внедрения сейчас)

## Risks / pitfalls

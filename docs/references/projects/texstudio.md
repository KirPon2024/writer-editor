## Summary (1–3 bullets)
- Большой “профессиональный” редактор (LaTeX) с сильными инструментами структуры и поиска.
- Полезен как референс UX: “структура документа”, панели, поиск, навигация.
- GPL → код не копируем.

## Relevant parts (where to look)
- В основном интересны UX‑паттерны из приложения (скрины/мануал), а не код.

## MVP Now (что можно взять прямо сейчас)
- Идеи UX для “Structure view” и навигации по документу.

## Later (идеи на будущее, без внедрения сейчас)
- “Глобальная структура” (книга/проект) + быстрые переходы.
